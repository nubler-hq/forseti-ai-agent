# Forseti - AI Browser Agent

**Forseti** é uma extensão de navegador para Google Chrome que transforma comandos de linguagem natural em ações de controle do navegador, utilizando o poder da API Gemini (Google AI).

## Funcionalidades
- **Controle de Navegador por IA:** Navegue, clique e preencha formulários usando comandos de voz ou texto.
- **Integração Nativa Gemini:** Utiliza a API Gemini para tradução de comandos.
- **Design Minimalista:** Interface de usuário limpa e inspirada em agentes de IA modernos.
- **Configuração Segura:** Armazenamento seguro da chave de API via `chrome.storage`.
- **Suporte Multimodal (Roadmap):** Planejamento para processamento de vídeos do YouTube e áudio.
- **Suporte MCP Servers (Roadmap):** Planejamento para integração com o Model Context Protocol.

## Roadmap
O desenvolvimento de Forseti será rastreado através das Issues do GitHub.

## Instalação
1. Clone este repositório.
2. Navegue para `chrome://extensions` no Chrome.
3. Ative o **Modo Desenvolvedor**.
4. Clique em **Carregar sem compactação** e selecione a pasta do repositório.
5. Configure sua chave de API Gemini na página de Opções.
